## Running w dev

- Clone repo.
- First run `code --install-extension coq-lsp-0.1.9-dev.vsix` to have the right coq-lsp.
- Open directory in vscode, F5 to launch extension dev host, open desired coq file, run command `vizcar.lspRender`.
